Selector de Atributos

Selector universal - *
* { font-family: 'Franklin Gothic Medium'; }
Selector por Etiqueta - NAV
nav { color: blue; }
Selector por class - P

.p { color: red; }

Selector por Id - P

#p { color: green; }

Selectores de atributo []

Selec.Element a-[href]
[href] { color: green; }
Selec.Element a-[href]=# Seleccionar los elementos que tienen ese atributo y ese valor
[href="#"] { color: red; }
Selec.Element b-[class~="verde] Seleccionar los elementos que tienen ese atributo "al menos ese valor una vez".
[class~="verde"] { color: green; }
Selec.Element b-[class|="blue"] Seleccionar los elementos que tienen ese atributo y exactamente ese valor o empieze por el valor seguido de un guión
[class|="verde"] { color: blue; }
Selec.Element b-[class^="Oscuro_rojo"] Seleccionar los elementos que tienen ese atributo y EMPIESA por ese valor
[class^="ABCrojo"] { color: white; }
Selec.Element b-[class*="rojo"] Seleccionar los elementos que tienen ese atributo y contengan ese valor
[class*="ABCrojoDFG"] { border: 3px solid blue; }
Selec.Element b-[class$="Oscuro_rojo"] Seleccionar los elementos que tienen ese atributo y TERMINA por ese valor
[class$="rojoABC"] { color: yellow; }

Selectores combinadores

Selector Hermano adyacente h1+h2 selcciona Subtítulo h2 hermano menor de h1.

h1

---h1+h2

h2

Selector Hermano general h1~h3 Selecciona TODOS los hermanos que esté por el y debajo de el . El estilo se aplica a todos los hermanos hermano menor de h1.

h1

---h1+h2

h2

Selectores Descendentes
-p
--samp Selecciona todos los hijos (QUE ESTEN DENTRO DE CONTENEDOR). El estilo se aplica a todos los hijos

pppp
p samp { color:red;}

Selector de hijo directo
-p
--span Selecciona todos los hijos que sean directos, es decir que no estén dentro de otras etiquetas. El estilo se aplica a todos los hijos

pppp
p>span { color:blue;}
p>span { color:blue;}
